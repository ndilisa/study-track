# Study-Track Web Platform

A simple student resource web platform designed to help university students access study materials, select courses, and receive resources via email.

## Features
- Login page
- Sign-up page for new users
- Resource selection by course
- Submission confirmation messages
- Clean and simple UI

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project was built to simulate an online study resource platform.

## What I learned
- Structuring multi-page websites
- Creating user navigation flows
- Handling form interactions with JavaScript
- Basic UI design principles

## Future Improvements
- Real authentication system
- Email API integration
- Backend with database
- Mobile responsiveness
